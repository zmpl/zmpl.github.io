    jQuery笔记

    jQuery 很容易学习。

    jQuery 是一个 JavaScript 库。

    jQuery 极大地简化了 JavaScript 编程。

    jQuery是一个轻量级的”写的少，做的多”的JavaScript库。

    基础语法:

    $(selector).action(),$表示JQuery,selector表示选择器,action表示执行的动作。

    ###jQuery选择器

    • 基本选择器

      * 元素选择器:`$("div")`选择所有的div元素。
* id选择器:`$("#one")`选择id=one的元素。
* 类选择器:`$(".two")`选择class=two的所有元素。
* 并集选择器:`$("selector,selector1,...,selectorn")`选择所有selector元素，所有selector1元素，...，所有selectorn元素。
* 全局选择器:`$("*")`选择所有元素。

    • 层次选择器

      * 后代选择器:`$("div span")`获取div后代中所有span元素。
      • 子类选择器:$("div>span")获取div直接子类中所有span元素。
      • 相邻元素选择器:$("div span")获取div的下一个元素，span元素。
      • 同辈选择器:$("div~span")获取div之后的所有span元素。

    • 属性选择器:

      • [attribute]:$("[href]")获取包含href属性的所有元素 。

      • [attribute=value]:$("href='http'")获取所有href=”http”的元素。

      • [attribute!=value]:$("href!='http'")获取所有href!=”http”的元素。

      • [attribute^=value]:$("href^='bc'")获取所有href=”abc”,href属性=bc结尾的元素。

      • [selector] [selector1] […] [selectorn]:$("li[id][href='http'][width='100px']")获取含有id属性并且href=”http”,width=”100px”的所有li元素。

    • 基本过滤选择器:

      • :first 选择第一个元素 $("div:first") 选择第一个div元素
      • :last 选择最后一个元素 $("div:last")选择最后一个div元素
      • :even 选择所有下标为偶数的元素(下标从0开始) $("div:even")选择下标为偶数的所有div元素
      • :odd 选择所有下标为奇数的元素(下标从0开始) $("div:odd")选择下标为奇数的所有div元素
      • :eq(index) 选择所有id=index的元素 $("div:eq(1)")选择下标为1的div元素
      • :gt(index) 选择所有id>index的元素 $("div:gt(1)")选择下标大于1的所有div元素。
      • :lt(index) 选择所有id<index的元素 $("div:lt(2)")选择下标小于2的所有div元素。
      • :not(selectors) 选择不包括selectors的所有元素，selectors可以是多个，用,隔开。 $("li:not(:first)")选择除开第一个的所有li元素。

    • 内容过滤选择器:

      • :contains(text) 选择元素内容包括text的所有元素。 $("div:contains('hello')")选择包含hello的所有元素。

      • :empty 选择内容为空的所有元素。 $("div:empty")选择为空的div元素。

      • :has(selctor) 选择含有选择器匹配的所有元素的元素。 $("div:has(span)")选择div含有span的所有span元素。

      • parent 匹配含有内容或文本的元素。 $("div:parent")选择含有内容的div元素。

    • 可见性过滤选择器

      • :hidden 选择所有隐藏元素 $("div:hidden")选择所有隐藏的div元素。
      • visible 选择所有显示元素 $("div:visible")选择所有可见的div元素。
